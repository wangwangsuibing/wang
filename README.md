## 🚗 核心模块：ADAS 数据采集平台（含源码）

对标行业竞品（dSPACE AUTERA、Vector vMDM、华为八爪鱼）设计的一站式采集管理平台，覆盖**任务下发 → 车端采集 → 数据回传 → 质检 → 管理消费**全流程：

### 采集作业管理
- **地图作业**：Leaflet 地图（高德瓦片 + WGS-84→GCJ-02 纠偏）打点、绘制路线/电子围栏、轨迹热力图、覆盖缺口网格分析
- **任务管理**：任务创建/下发/进度跟踪，绑定车辆、采集员、传感器配置与事件触发规则（如 `AEB:10:30`）
- **出车检查单**：下发前 6 项 SOP 强制门禁（标定/镜头/自检/天线/存储/时同）
- **Campaign 批量任务**：一套配置多车批量生成任务
- **车辆/采集员管理**：实时车辆状态模拟、轨迹回放、采集员指派与工作量统计

### 数据包管理
- **分片断点续传上传**：2MB 分片 + 会话恢复 + SHA-256 校验 + 去重 + 传感器自动分类
- **可配置质检**：8 项检查（完整性/丢帧率/时间同步/相机曝光/点云密度/GNSS 失锁等），阈值在线配置，不合格**一键生成补采任务**
- **数据治理**：合规脱敏标记、优先级回传调度、标签检索、数据血缘（下游消费记录）、保留策略自动归档、manifest/ZIP 导出
- **运营支撑**：近 14 天趋势报表、存储水位监控、操作审计日志、告警中心

### 界面（对标 Ant Design Pro / Samsara 交互规范）
- 左侧可折叠导航（7 个模块）；实时监控页全屏地图 + 浮动 KPI 卡
- 任务/数据为标准表格列表页（筛选栏 + 操作列）；创建表单与详情均为右侧 Drawer

### 技术栈
- **后端**：Python + FastAPI + SQLite（WAL），车辆移动/任务进度/数据回传模拟器
- **前端**：原生 HTML/CSS/JS（零框架依赖），Leaflet 地图
- **运行**：`cd adas-data-platform && run.bat`，自动装依赖并启动 http://127.0.0.1:8080

## 🖥 其他模块

- **数据闭环平台**（`数据平台.html`）：WorldSim/LogSim 场景资产管理，标签组合检索、轨迹预览、打标、xosc/xodr 下载，一键导出仿真
- **仿真测试平台**（`仿真平台.html`）：卡片式用例管理、多条件筛选、运行状态标识、测试结果反哺场景挖掘

两者均为纯静态单文件 HTML，双击即可用 Chrome/Edge 打开体验。

## 🚀 快速开始

```bash
git clone https://github.com/wangwangsuibing/wang.git
cd wang/adas-data-platform
run.bat        # 启动采集平台 → http://127.0.0.1:8080
```

静态页面直接双击 `数据平台.html` / `仿真平台.html` 打开。
