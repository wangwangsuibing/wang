# 王嘉伟作品集 Demo

自动驾驶仿真测试领域的界面设计作品精选（Demo 版），均为纯静态 HTML 单文件，无需安装任何依赖，**双击浏览器直接打开即可查看**。

## 文件说明

| 文件 | 内容 | 说明 |
|------|------|------|
| `仿真平台.html` | 仿真测试平台 · 我的用例（卡片视图） | 自动驾驶仿真测试平台的用例管理界面，包含用例卡片、筛选、状态标签等交互 |
| `数据平台.html` | 数据平台 · 场景管理 | 仿真数据资产平台的场景管理界面，暗色主题，包含场景列表与资产浏览 |

## 使用方式

1. 直接双击任意 `.html` 文件，在 Chrome / Edge 等现代浏览器中打开；
2. 页面内的按钮、下拉、切换等交互均由内联 JavaScript 实现，可直接体验；
3. 建议使用 1920×1080 及以上分辨率浏览以获得最佳效果。

## 技术说明

- 纯 HTML + CSS + 原生 JavaScript，单文件自包含，无外部依赖、无需网络；
- 设计遵循企业级 B 端产品规范：栅格布局、语义化色彩变量（CSS Custom Properties）、明暗主题。
