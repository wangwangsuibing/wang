## 🚗 自动驾驶方向

### 1. 仿真测试平台（`仿真平台.html`）

面向自动驾驶算法（AEB/ACC/LKA 等）仿真验证的测试平台前端 Demo：

- **用例管理**：卡片式展示仿真测试用例，覆盖场景缩略图、名称、创建时间等；
- **搜索与筛选**：多条件组合筛选，快速定位目标用例；
- **状态标识**：不同颜色标签区分用例运行状态（待运行/运行中/通过/失败）；
- **仿真流程**：新建、运行、编辑、删除等操作闭环，支撑算法回归测试。

### 2. 数据闭环平台（`数据平台.html`）

面向自动驾驶数据闭环（采集 → 挖掘 → 场景生成 → 仿真回灌）的场景管理平台 Demo：

- **场景资产管理**：WorldSim / LogSim 场景数据列表，支持列表与缩略图双视图；
- **智能检索**：标签组合检索、时间区间筛选、关键字搜索，支撑场景挖掘；
- **数据操作**：详情、可视化轨迹预览、手动打标、xosc/xodr 下载；
- **暗色主题**：企业级 B 端暗色视觉风格，适配长时间数据分析工作。

### 3. ADAS 数据采集平台（`ADAS数据采集平台-v1.0-win64.zip`）

Windows 桌面端路采数据采集工具 Demo：

- 面向 ADAS/自动驾驶路采场景的多传感器数据采集与管理；
- 解压后直接运行主程序（Windows 10/11 64 位）。

## 🚀 使用方式

1. 克隆或下载本仓库：
   ```bash
   git clone https://github.com/wangwangsuibing/wang.git
   ```
2. 双击任意 `.html` 文件，用 Chrome / Edge 打开即可体验（建议 1920×1080 及以上分辨率）；
3. 桌面程序：解压 zip 后运行主程序（Windows 64 位）。
