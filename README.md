## 🖥️ 项目介绍

### 1. 仿真平台（`仿真平台.html`）

面向自动驾驶算法测试的仿真平台前端 Demo，核心功能界面包括：

- **用例管理**：以卡片形式展示测试用例，支持缩略图、名称、创建时间等信息一览；
- **搜索与筛选**：顶部搜索框 + 多条件筛选，快速定位目标用例；
- **状态标识**：不同颜色标签区分用例状态，直观清晰；
- **操作交互**：新建、运行、编辑、删除等按钮均有 hover/点击反馈。

### 2. 数据平台（`数据平台.html`）

面向数据闭环的场景数据管理平台前端 Demo，核心功能界面包括：

- **场景管理**：场景数据列表展示，支持列表/缩略图两种视图；
- **数据检索**：标签组合检索、时间区间筛选、关键字搜索、一键重置；
- **暗色主题**：企业级 B 端暗色视觉风格，长时间使用不易疲劳；
- **数据操作**：详情查看、可视化预览、打标、下载等操作入口。

### 3. ADAS 数据采集平台（`ADAS数据采集平台-v1.0-win64.zip`）

Windows 桌面端数据采集工具 Demo：

- 解压 zip 后运行其中的可执行程序即可体验；
- 面向 ADAS/自动驾驶路采场景的数据采集与管理；
- 仅供演示，请在 Windows 10/11 64 位系统运行。

## 🚀 使用方式

**静态页面：**

1. 下载或克隆本仓库：
   ```bash
   git clone https://github.com/wangwangsuibing/wang.git
   ```
2. 直接双击任意 `.html` 文件，在 Chrome / Edge 等现代浏览器中打开；
3. 页面内按钮、下拉、视图切换等交互均由内联 JavaScript 实现，可直接体验；
4. 建议使用 **1920×1080 及以上分辨率** 浏览以获得最佳效果。

**桌面程序：**

1. 下载 `ADAS数据采集平台-v1.0-win64.zip` 并解压到任意目录；
2. 运行解压后的主程序（Windows 64 位）。

## 🛠️ 技术说明

- 纯 **HTML + CSS + 原生 JavaScript**，单文件自包含，无外部依赖、无需网络；
- 设计遵循企业级 B 端产品规范：
  - 栅格化布局，信息层级清晰；
  - 语义化色彩变量（CSS Custom Properties），支持明/暗主题；
  - 组件化视觉语言：卡片、标签、弹窗、分页等风格统一；
- 兼容 Chrome、Edge 等现代浏览器（不支持 IE）。
