# 王嘉伟作品集 Demo —— 自动驾驶 & 机器人通用全流程数据闭环

> 聚焦**自动驾驶与机器人数据闭环**方向的产品作品精选（Demo 版），完整覆盖 **数据采集 → 数据管理 → 仿真平台测试** 的全流程链路，方法论同时适用于**自动驾驶**与**机器人/具身智能**领域。
> 前端演示均为纯静态 HTML 单文件，无需安装任何依赖，双击即可在浏览器中打开体验。

## 🔄 全流程数据闭环

本作品集三个模块串联成一条完整的自动驾驶与机器人数据闭环链路：

```
┌──────────────┐      ┌──────────────┐      ┌──────────────┐
│  ① 数据采集   │ ───▶ │  ② 数据管理   │ ───▶ │  ③ 仿真测试   │
│ ADAS采集平台  │      │  数据闭环平台  │      │  仿真测试平台  │
└──────────────┘      └──────────────┘      └──────────────┘
   路采·真机采集         场景挖掘/打标           用例回归/评价
        ▲                                          │
        └────────── 问题场景反哺采集与算法迭代 ◀────┘
```

- **① 数据采集**：车端/机器人端多传感器数据采集，产出原始数据；
- **② 数据管理**：原始数据切片、Log2World 场景生成、标签化管理与检索挖掘；
- **③ 仿真测试**：场景导出至仿真平台生成用例，对自动驾驶与机器人算法（AEB/导航/避障/跟随等）进行回归测试；
- **闭环反哺**：测试暴露的问题场景反向指导采集与场景挖掘，驱动算法持续迭代。

## 🚗 模块介绍

### 1. ADAS 数据采集平台（`ADAS数据采集平台-v1.0-win64.zip`）

数据闭环的起点 —— Windows 桌面端路采数据采集工具 Demo：

- 面向 ADAS/机器人真机数据采集场景的多传感器数据采集与管理；
- 采集数据作为后续数据管理与场景生成的原始输入；
- 解压后直接运行主程序（Windows 10/11 64 位）。

**源代码**：见 `adas-data-platform/` 目录 —— Python（FastAPI 风格）后端 + HTML/CSS/原生 JS 前端，内置 Leaflet 地图（高德瓦片，含 WGS-84→GCJ-02 坐标纠偏）与采集数据模拟器；双击 `run.bat` 自动安装依赖并启动（http://127.0.0.1:8080）。

### 2. 数据闭环平台（`数据平台.html`）

数据闭环的中枢 —— 面向数据管理（采集数据 → 切片 → 场景生成 → 标签挖掘）的场景管理平台 Demo：

- **场景资产管理**：WorldSim / LogSim 场景数据列表，支持列表与缩略图双视图；
- **智能检索**：标签组合检索、时间区间筛选、关键字搜索，支撑场景挖掘；
- **数据操作**：详情、可视化轨迹预览、手动打标、xosc/xodr 下载；
- **仿真衔接**：场景可一键导出到仿真平台生成测试用例；
- **暗色主题**：企业级 B 端暗色视觉风格，适配长时间数据分析工作。

### 3. 仿真测试平台（`仿真平台.html`）

数据闭环的验证端 —— 面向自动驾驶与机器人算法（AEB/导航/避障/跟随等）仿真验证的测试平台前端 Demo：

- **用例管理**：卡片式展示仿真测试用例，覆盖场景缩略图、名称、创建时间等；
- **搜索与筛选**：多条件组合筛选，快速定位目标用例；
- **状态标识**：不同颜色标签区分用例运行状态（待运行/运行中/通过/失败）；
- **测试闭环**：新建、运行、编辑、删除等操作闭环，测试结果反哺场景挖掘与算法迭代。
